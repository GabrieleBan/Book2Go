Front-end sviluppato con Node.js, ShadCN e Taiwind.
L'applicazione fornisce l'interfaccia per un back-end strutturato a microservizi di una organizzazione a "catena" di librerie che anche servizi online (come prestiti digitali, prenotazioni) agli utenti (diversi tier abbonamento).

La presentazione inserita fornisce una panoramica delle funzionalità offerte dall'organizzazione (parti implementate indicate) con la relativa documentazione e architettura.
